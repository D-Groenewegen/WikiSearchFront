# WSSearchFront

a vue.js frontend for WSSearch

Build version! for 1.31 and 135

example
```
{{#searchEngineConfig:Class=Page|Version|Title|?Title}}

{{#loadSearchEngine:size=2
|?Title=combobox
|?Modification date=date
}}

```


```
{{#searchEngineConfig:Class=Page|Version|Title|?Title}}

{{#loadSearchEngine:size=2
|?Title=combobox
|?Modification date=datepicker
}}

```
